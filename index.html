Welcome to Git & GitHub Tutorial

  • Introduction

    Open the directory in terminal and Initialize the empty git repository. git init .It will create a .git directory in the current directory.

  • Git Status

    Now one of the useful command here is git status. It will tell you about what files are modified, added or deleted

  • Git Add

    Now it is time to add the files to staging area (Tracking changes). for that we'll use the command : git add (filename) for specific file or git add . . for adding all files

  • Git Commit

    Once the files are in staging area you can save them to commit history via command git commit -m "My First Commit"

  • Git log

    you can check all your commit from the command git log or git log --oneline

  • Upload on GitHub

    ow upload these changes to GitHub. for that we have to link our local repository to github repository. git remote add origin URL ( URL will be provided by gitHub ).

  • Push code on gitHub

    Now the next step is to push the code on git hub.To push your code on gitHub we use the following cmmand

    git push -u origin main / branch-name
  • Git Pull

    Git pull command is used to pull the code from gitHub to your local repository and to make your local and online repository at the same level

    git pull

    • Branching........

  • Create & move to new branch

    The best feature of the git is to create a new branch where we can add new feature and test them without distubing the main code. For this we can use the following command

    git checkout -b new-feature
  • Return to main Branch

    To return to main branch we have to use the following command. git checkout main

  • Delete a Branch

    Now if we want to delete a branch we can use the following code

    git branch -d new-feature
